This Directory contains the following tools that shall ease 
the work with ldf (Lin Description File) and CANalyzer/ CANoe:

- ldf2dbc.exe
  This converter takes a *.ldf and produces a *.dbc.
  You will find the documentation in the help system of CANoe/CANalyzer.
  This executable uses the DLL ldf2dbc.dll 
- ImportLDF.exe 
  This application integrates the ldf2dbc converter in CANdb++.
  You will find the documentation in the help system of CANdb+.
